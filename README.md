# Ярослав Шинкарёв — личный бренд

Лендинг для получения заявок на создание сайтов, Telegram-воронок и консультаций.

**Цель:** за 5 секунд посетитель понимает, кто такой Ярослав, чем помогает,
почему стоит написать и как оставить заявку.

## Структура
- `index.html` — разметка лендинга (один экран = один вопрос пользователя)
- `styles.css` — оформление: тёмный минимализм, воздух, микроанимации
- `script.js` — плавная навигация, reveal-анимации, параллакс, форма заявки
- `assets/` — место для фотографии (`yaroslav.jpg`)

## Секции
1. Hero — оффер, кнопки, мини-факты, фото
2. Обо мне — 3 карточки (Кто я / Что умею / Что получает клиент)
3. Услуги — Сайт под ключ · Telegram-воронка · Консультация
4. Почему со мной работают — 5 преимуществ
5. Мои проекты — портфолио-карточки
6. Почему мне доверяют — путь от изучения до запуска
7. CTA + форма заявки с анимацией успеха
