# Фотографии сайта

- `yaroslav.jpg` — оригинальное фото Ярослава (полный кадр).
- `yaroslav-hero.jpg` — кадрированный портрет, используется в Hero и блоке «Доверие».

## Как заменить фото
1. Положите новое фото в эту папку (вертикальное, формат ~3:4 или 4:5).
2. Замените `yaroslav.jpg` и при необходимости пересоздайте кадр `yaroslav-hero.jpg`.
3. Позиционирование кадра настраивается в `styles.css`
   (`.hero__photo--filled` и `.trust__photo.trust__photo--filled`, свойство `background-position`).
