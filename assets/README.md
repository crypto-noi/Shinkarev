# Фотография для сайта

Добавьте сюда фото Ярослава с именем **`yaroslav.jpg`** (вертикальное, формат ~4:5).

Затем в файле `index.html` найдите блоки с классом `hero__photo` и `trust__photo`
и замените плейсхолдер на фон-картинку. Например, для hero:

```html
<div class="hero__photo" id="heroPhoto"
     style="background-image:url('assets/yaroslav.jpg')"></div>
```

Картинка автоматически впишется в дизайн (object-fit / cover уже настроены в CSS).
